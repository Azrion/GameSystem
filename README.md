<i>

<ol>
    <li>Install dependencies by executing in terminal:

    <li>Run <b><i>__main__.py</i></b> to start the program</li>
    <li>Run <b><i>pseSimulator.py</i></b> as a separate process to simulate messages from the PSE system</li>
